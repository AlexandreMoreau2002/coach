# /coach

Utilise le skill `coach:coach` comme point d'entree.

Respecte sa sequence de demarrage :

1. Lire silencieusement le contexte du carnet Obsidian.
2. Invoquer `analyse-seances` pour les dernieres seances.
3. Poser le check-in en 3 questions, une par une.
4. Router vers creation de seance, reflexion methodologique ou planification selon les reponses.

Ne remplace pas les consignes du skill. Charge `coach:coach`, puis applique ses instructions.
